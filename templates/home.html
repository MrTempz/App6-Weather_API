Weather data API

URL format for station and date: http://127.0.0.1:5000/api/v1/station/date

URL example: http://127.0.0.1:5000/api/v1/10/2012-12-21


URL format for all historic data for station: http://127.0.0.1:5000/api/v1/station

URL example: http://127.0.0.1:5000/api/v1/10


URL format for yearly data for station: http://127.0.0.1:5000/api/v1/yearly/station/year

URL example: http://127.0.0.1:5000/api/v1/yearly/10/1988
